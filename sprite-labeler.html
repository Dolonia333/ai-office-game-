Sprite Labeler

Label sprites from Modern Office sheets. Pick a sheet, then set a Catalog ID (e.g. desk_pod, chair_office, water_cooler, printer) and Type for each. Leave ID blank to omit from catalog. Then click Save catalog to download the corrected JSON.

Reference checklist

Pick a required ID, then click a sprite card and press Assign. This is the “game-builder” workflow: checklist → place exact props.
Tip: you can Alt+click a card to assign instantly when a required ID is selected.
